博客创建笔记

发表于 2016-04-18   |   分类于 互联网   |  

如何从零开始创建一个个人博客。

我为什么要建博客？

因为我想要变得不一样。

某天夜里，我从恶梦中惊醒，却失去了再次入睡的能力，打开手机，透过虎扑看曾经中国人的主队火箭被勇士吊打，无趣。然后开始胡思乱想，我不知道自己未来会怎样。20出头的年纪，却好像是已经死了，只不过等几十年再埋而已。回想过去，似水流年，真正留下来的就仿佛真的只是沉浸在骨子里。慢慢的，经历越来越多，留下来的记忆却越来越少，从一开始的高清视频，逐渐压缩，最终蜕变成一张张黑白相片。甚至，化为虚无，连一丝痕迹都没有留下。记忆越来越短暂，三五年后，我还能记住现在的我么，也许只是在记忆中留下了简单的一笔，最终消散在时光里。

如果我现在死去了，世界有什么能说明我活过，除了那些我认识的人，谁还能去认识一个我。或者说，十年后，我还能记得我自己这一路走过的路么，我还能记得我曾经努力过奋斗过的那些梦想么？所以，我想要找一个平台，留下我的一些痕迹，学习的笔记也好，小说散文也罢，包括将来的许多许多可能性，让它帮我记录我的一段段人生旅程。

也许有一天，当有人想要认识我的时候，可以不用去向她或他介绍我几十年的过往，只需要给出一个链接，看看我一路走来，了解我的为人，去辨别是否值得结交。或者，未来，给我的孩子们讲述我的过往时，不需要那么辛苦的拼凑回忆，可以慢慢地和他们讲我一路的风景。

博客建立过程

整个个人博客建立，大概花费了四天时间，包括购买域名，学习github pages和hexo搭建博客，网络上有许多教程，在此就不再赘述，也是因为自己并不是很熟悉，等之后把这些方法玩的熟悉一些，可能会详细的写一个教程，就像我借助前人的文档一样。

一下是几个主要的步骤。

1. 购买域名：jonzhang.me

域名网站：godaddy

时间：2016年4月17日星期日

记录：试图开发一个个人博客空间，博客中主要分享自己的学习笔记、随笔等。博客的定位比较模糊，目前而言，内容可能比较杂乱。既包括技术笔记，也包括读书笔记，心情随笔等。尽可能提高质量，但是更主要是作为互联网时代的一份自我的记忆，个人博客的时代已经过去，但是我还是需要一个地方来留存自己的一些过往。

2. Github pages

使用github配置独立博客参考上述文章，学习了建立方法。基本搭建完成。

其中遇到了几个问题

  1. Ssh key 生成后，最好是存储与原有的路径下，自作聪明的新建了一个文件夹存储，导致后面出错
  2. __Config文件，配置ssh端口为 port 22，后改为443, 参考
    http://blog.163.com/zhou_411424/blog/static/197362156201281725033556/

    Godaddy绑定域名后，可以用绑定的域名 jonzhang.me访问 sweeneyzhang.github.io，但是页面显示的域名还是 sweeneyzhang.github.io
    需要自定义域名为 jonzhang.me
  4. 使用Hexo框架实现个人博客，参考以下链接
    http://zipperary.com/2013/05/29/hexo-guide-3/
    选择主题为Alex，来源 https://github.com/hexojs/hexo/wiki/themes
  5. Hexo本地部署启动失败解决方案，无法使用 localhost:4000 查看本地部署
    解决方案：
    http://byenow.top/2016/01/10/Hexo%E6%9C%AC%E5%9C%B0%E9%83%A8%E7%BD%B2%E5%90%AF%E5%8A%A8%E5%A4%B1%E8%B4%A5%E8%A7%A3%E5%86%B3%E6%96%B9%E6%A1%88/
  6. 使用Alex和NexT主题后，选择NexT主题。参考next的使用说明，设计了about和标签、分类等页面。添加了社交网站。

内容为王

一个博客，重要的是其内容，形式外表做的再华丽，如果没有足够的内涵，也不会长久存在，如果我不能为读者提供干货，自己会失去继续维护博客的动力。
后面着重于博客内容，形式基本确定，只需要小改动或者增加一些功能，后期慢慢完善。
