Single Number

题目简述

Given an array of integers, every element appears twice except for one. Find that single one.
Note: Your algorithm should have a linear runtime complexity. Could you implement it without using extra memory?

Python 程序

这个题目是要求出所给的一个数列中仅出现一次的数值，难点在于需要设计一个时间复杂度为n的算法，同时不适用额外的内存空间。
除了最终返回的数字外，其他数字都出现了两次。根据异或操作的特性，两个相同的数值异或后等于0，一个数与0异或得到其本身。所以可以对数列逐个异或得到最终的Single number。

总结

对于位操作，一向是接触比较少的，在思维中总是会不经意遗忘这一类高效、简单的运算。位操作主要包括以下几种：按位与(&)、按位或(|)、按位取反(~)、按位异或(^)以及移位操作(<<，>>)。本题目中借助了异或的特性：一个数与他本身异或得到0，与0异或得到他本身。数列中，所有出现两次的数字异或之后为0，仅出现一次的数与0异或，得到他本身，所以能够计算得到这个数列中的“单身狗”。执行完这个程序，需要对数列遍历一次，而内存仅需要存储一个整形的变量。

Sum of Two Integers

题目简述

Calculate the sum of two integers a and b, but you are not allowed to use the operator + and -.
Example: Given a = 1 and b = 2, return 3.

Python 程序

题目中要求，不能使用加减法来实现两整型数的相加。如果不能使用加减法，那么只能通过位运算来实现了求两数和了。对于整数而言，两数的按位与左移一位即可得到每一位的进位，按位异或可以得到每一位的和，再将进位与按位和进行下一次计算，重复以上操作，直到进位为0，输出和数即可。

总结

使用位运算实现加法，需要考虑两个问题，一个是整型数的位宽，一个是负数的运算。所以在实现过程中，需要考虑溢出以及最后取值的符号。
